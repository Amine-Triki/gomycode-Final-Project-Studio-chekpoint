## Overview
The Todo App is a simple web-based application designed to help users manage daily tasks.  
The project focuses on practicing frontend fundamentals using pure HTML, Tailwind CSS, and Vanilla JavaScript.

## Goals
- Practice DOM manipulation with JavaScript
- Build a clean and responsive user interface
- Understand event handling and basic app logic

## Functional Requirements
- Users can add new todo items
- Users can mark todo items as completed
- Users can delete todo items
- Empty todo items are not allowed

## UI / UX Requirements
- Simple and intuitive interface
- Responsive layout for mobile and desktop
- Styled using Tailwind CSS

## Technical Requirements
- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript only (no frameworks or libraries)
- Runs entirely in the browser

## Non-Functional Requirements
- Lightweight and fast
- Easy to read and maintain code
- Cross-browser compatibility (modern browsers)

## Future Enhancements
- Persist todos using Local Storage
- Edit existing todo items
- Filter todos (All / Active / Completed)
- Dark mode support

## Target Audience
- Beginner frontend developers
- Students learning JavaScript and web development
